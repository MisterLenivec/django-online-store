Об этом проекте

Проект не является коммерческим (тоесть не оказывает услуг по продаже товаров), создан для освоения технологий разработки web-приложений.

Проект является интеренет-магазином, предоставляющим товары. В данном случае электронику.

Можно выбирать товары, добавлять в корзину, применять купоны на скидки, оформлять заказ. Оплата отключенна. После заказа присылается сообщение по электронной почте, с деталями заказа, а так же PDF файл. Реализованна регистрация и авторизация, профиль пользователя с возможностью просмотра заказов и изменением аватара пользователя.

  • Проект создавался с помощью веб-фреймворка Django 2 (Python).
  • Взаимодействие с базой данных осуществляется посредством механизма объектно-ориентированного отображения Django ORM.
  • В качестве СУБД была использованна свободная объектно-реляционная СУБД PostgreSQL.
  • Клиентская сторона пользовательского интерфейса написанна на HTML/CSS с использованием Flexbox и Grid CSS (Сайт адаптивный).
  • На javascript написан всего один не большой скрипт.
  • Система контроля версий Git.
